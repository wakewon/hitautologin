# 更新历史 (History)

## v1.3.2 (2026-01-04)
- **新增功能**：新增系统更新弹窗自动点击刷新功能（自动检测并点击"立即刷新"按钮）。
- **新增功能**：新增页面 Ant Design Float Button 自动位置调整，避免与 HIT 悬浮按钮重叠。
- **新增功能**：使用 MutationObserver 持续监控动态加载的悬浮按钮，确保自动适配。
- **修复 Bug**：绕过 `mail.hit.edu.cn` 的自动登录功能，解决兼容性问题（Issue #4）。
- **同步**：所有平台代码同步更新（Chrome / Edge / Firefox / Tampermonkey）。

## v1.3.1 (2025-12-03)
- **修复 Bug**：修复了在 Firefox 和 iOS Safari 浏览器中，页面底部会出现不应显示的纯文字内容的问题。
- **重构**：Tampermonkey 脚本的悬浮按钮（FAB）和遮罩层重构为使用 **Shadow DOM**，彻底隔离样式，避免与页面样式冲突。
- **优化**：Chrome 扩展优化了 UI 元素的注入位置（从 `documentElement` 移至 `body`），提高了兼容性。
- **新增平台**：发布 **Firefox 扩展版本**，已上架 Firefox Add-ons 商店，功能与 Chrome 插件版完全一致。
- **新增平台**：发布 **Microsoft Edge 扩展版本**，已提交至 Edge Add-ons 商店审核。

## v1.3.0 (2025-11-21)
- **新增功能**：支持 WebVPN 访问重定向，方便在校外访问内网资源。
- **新增功能**：新增校外访问授权（IDP）页面的自动同意功能，减少点击操作。
- **UI 升级**：悬浮按钮（FAB）界面全新升级，支持暗色模式适配，增加流畅动画。
- **同步**：实现了 Tampermonkey 脚本与 Chrome 扩展的功能对齐。

## v1.2.2 (2025-11-04)
- **Chrome 扩展**：新增设置实时同步功能，修改配置无需刷新页面。
- **Chrome 扩展**：悬浮按钮支持动态刷新显示。
- **Chrome 扩展**：内置帮助页面（help.html）。

## v1.2.1 (2025-11)
- **Tampermonkey**：新增悬浮入口（FAB）功能，提供快捷操作面板。

## v1.0.0 (2024-09)
- **初版发布**：实现核心的自动填充与自动登录功能。
- **发布**：Tampermonkey 脚本发布至 GreasyFork。
